# 🌤️ CityWeatherApp

This is an extremely simple weather app (.NET MAUI Blazor),
quickly made as a means to experiment with API consumption in mobile development.

It allows any user to search a city (or any location registered in the geocoding API) and see the weather forecast for the moment.
Also, whnen opening, it shows the weather at the current location, provided the user has granted permission.

---

##  ✨ Features

- Current location weather description (if permitted by the user)
- Location search by name and country (optional)
- Weather description of the selected location
